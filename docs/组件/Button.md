# Button 按钮

封装 element 的 el-button 组件，可以使用其所有的属性与方法，

特点:

- 当使用 Api 请求时，添加 `loading:true`属性，该按钮讲自动开启 loading，并且自动关闭。

- 二次确认。

## Attributes 说明

|  参数   | 说明                   |  类型   | 默认值 |
| :-----: | :--------------------- | :-----: | :----: |
| confirm | 点击按钮时给予二次确认 | Boolean | false  |

## Events 说明

| 事件名称 | 说明             | 回调参数 |
| :------: | :--------------- | :------: |
| callback | 点击按钮触发回调 |    无    |
